# Áurea Print Urovo v0.2 · Return Bridge

APK puente para Urovo i9100 con impresora térmica interna.

## Nuevo en v0.2

- Recibe `aureaprint://print?text=...&returnUrl=...&autoReturn=1`.
- Imprime el ticket.
- Regresa automáticamente a la página de AUREA que envió la impresión.

## Compilar sin PC potente

1. Sube este proyecto a GitHub.
2. Entra a Actions.
3. Ejecuta Build Áurea Print APK.
4. Descarga `app-debug.apk`.
5. Instálalo en el Urovo.

Si Android dice que no se pudo actualizar por conflicto, desinstala la app Áurea Print anterior y vuelve a instalar esta. No se pierde nada importante.
